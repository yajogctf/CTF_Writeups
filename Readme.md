# Outils et Write-ups

Vous trouverez ici quelques WU de challenges du site [Hackropole](https://hackropole.fr/fr/), site qui regroupe les challenges proposés chaque année lors du CTF France Cybersécurity Challenge (FCSC).  
Pas mal de cryptographie, et plutôt d'un niveau bas pour commencer ...  

Dans la section `Outils`, j'y insère des scripts python qui sont assez classiques et qui servent régulièrement dans les challenges (chiffrements César, Vigenère, Théorème Chinois, etc.).
